# Scripts

See `docs/PROJECT_STRUCTURE.md` for the full map.

| Folder | Role |
|--------|------|
| `autoload/` | Singletons registered in `project.godot` |
| `core/` | Shared utilities (`lane_coords.gd`) |
| `entities/` | Tower, creep, projectile, core, builder |
| `match/` | Match scene controller + camera |
| `systems/` | Lanes, pathing, waves, economy, AI |
| `ui/` | Screens; `components/` = reusable widgets |
| `vfx/` | Visual effects |
